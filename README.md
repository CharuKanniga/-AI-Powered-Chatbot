**AI Powered Chatbot (Chatbase)**
 **Overview**

This project is an AI-powered chatbot built using Chatbase, which internally follows the Retrieval-Augmented Generation (RAG) approach. The chatbot retrieves relevant information from an uploaded knowledge base and generates accurate, context-aware answers to user queries.

** Features**

Natural language question answering

RAG-based responses using uploaded data

Automatic information retrieval

Secure user identification (JWT)

Easy web integration

**Tools Used**

Chatbase (Managed RAG chatbot platform)

AI Language Models

JWT Authentication

Cloud deployment (learning level)

** Purpose**

To provide quick, accurate answers from documents and reduce manual information searching.
